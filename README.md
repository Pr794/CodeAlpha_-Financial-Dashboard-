## 📌 Project Overview
This project is a **Financial Health Dashboard** built in **Power BI** as part of my **Power BI Internship at CodeAlpha**. The dashboard provides a consolidated, interactive view of personal/business finances — tracking income, expenses, savings, net worth, and cash flow trends over time.

The goal of this project is to transform raw financial data into clear, actionable insights that support better budgeting and financial decision-making.

---

## 🎯 Objectives
- Consolidate key financial metrics (Income, Expenses, Savings, Net Worth) into a single view
- Visualize budget vs. actual spending across categories
- Track profitability trends with forecasting
- Monitor balance sheet health (assets vs. liabilities) over time
- Enable interactive filtering by category and month

## 🛠️ Tools & Skills Used
- **Power BI Desktop** — Data modeling, DAX measures, and visualization
- **Data Visualization** — KPI cards, trend lines, bar charts, area charts
- **Forecasting** — Trend-based projection for cash flow
- **Interactive Filtering** — Slicers for Category and Month

---

## 🚀 Key Insights
- Net cash flow shows a consistent upward trend from mid-2024 into early 2025
- Housing and Salary represent the largest categories in the budget vs. actual comparison
- Total assets consistently outweigh liabilities across the tracked months, indicating positive financial health
